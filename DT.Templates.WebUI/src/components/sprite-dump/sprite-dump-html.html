Sprites generated by Gulp

  • These sprites are generated by Gulp on build via the 'sprite' build task. Inspect the elements to see how to add them to your pages. You can have as many sprite sets as you want by modifying the number of folders in /src/sprites. You just have to manually include the css file generated for these sprites in the /Views/Shared/_Layout.cshtml file
